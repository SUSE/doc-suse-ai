= {productname} installation flowchart

:override-title!:

The following chart illustrates the installation process of {productname}.
ifdef::deployment_standard[]
It lists all the steps you have to perform if you prefer the *manual* installation of each component.
If you prefer automating the installation of selected steps, refer to xref:ai-automated-deployments[] for more details.
endif::[]
The chart outlines the following possible scenarios:

* You have clean cluster nodes prepared without a supported Linux operating system installed.
* You have a supported Linux operating system and {kube} distribution installed on cluster nodes.
* You have {rancherprime} and all supportive components installed on the {kube} cluster and are prepared to install the required applications from the {ailibrary}.
