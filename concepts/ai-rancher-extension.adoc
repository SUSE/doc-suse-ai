[#ai-rancher-extension-roles]
== Workflows and responsibilities

The workflow is defined by a user role and is different for a user and an administrator.

Administrator::
  * Installs {sairancherextension}.
  * Manages users and user roles.
  * creates ClusterRepo repositories.
User::
  * Deploys AI applications using the {sairancherextension}.
